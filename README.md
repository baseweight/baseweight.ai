# Baseweight Website

This is a Next.js App that runs the Baseweight website.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Configure environment variables:
   - Copy `.env.example` to `.env.local`
   - Add your SendGrid API key (required for contact form)

3. Run the development server:
```bash
npm run dev
```

## Environment Variables

- `SENDGRID_API_KEY` - Required for the contact form to send emails. Get your API key from [SendGrid](https://app.sendgrid.com/settings/api_keys)
